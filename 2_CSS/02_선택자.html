1. 전체 선택자 : *

현재 해당 페이지의 모든 태그들을 다 선택하고자 할 때 사용된다.

2. 태그 선택자 : 태그

현재 해당 페이지에 해당 태그들을 다 선택하고자 할 때 사용된다.

3. 클래스 선택자: .클래스명

해당 페이지에 내가 원하는 요소 "여러개" 선택하고자 할 때 사용된다.

  • 클래스선택자1
  • 클래스선택자2
  • 클래스선택자3
  • 클래스선택자4
  • 클래스선택자5

4. 아이디 선택자 : #아이디명

현재 해당 페이지에 특정 html 요소 "하나만"을 선택하고자 할 때 사용

5. 그룹 선택자

콤마(,)를 사용하여 여러 요소를 선택할 때 사용

div 태그입니다.

p 태그입니다.

6. 자식선택자와 자손선택자

요소들이 중첩되어 작성 가능
자식(>) : 바로 하위인 요소들
자손(띄어쓰기) : 하위 요소들 전부 다

현 div의 자식이면서 자손이다(바로 하위인 요소임과 동시에 하위 요소 그 자체이기 때문)

  • ul의 자식이면서 div의 자손이다(div의 자식은 아니다)
    • 바로 위 ul의 자식이면서 div의 자손이다
    • 바로 위 ul의 자식이면서 div의 자손이다
